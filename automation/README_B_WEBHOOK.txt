Как развернуть Вариант B (красивый UI + отправка в Google Sheets через веб‑хуκ)

Шаги в Google Apps Script (русский интерфейс):
1) Откройте https://script.new под вашим Google‑аккаунтом.
2) Создайте пустой проект → вставьте файл `cfg_webhook.gs` из папки automation/google-apps-script.
3) Вверху выберите функцию `setConfig` и нажмите ▶ Выполнить:
   setConfig('C.F.G.consulting@bk.ru', 'ВАШ_ТОКЕН_БОТА', 'ВАШ_CHAT_ID');
   (token/chat_id можно не указывать, если Telegram не нужен. Разрешите доступы.)
4) Нажмите **Развертывание → Управление развертываниями → Новое развертывание**.
   - Тип: **Веб‑приложение**.
   - Выполнять от имени: **Я**.
   - Кто имеет доступ: **Все** (или «Все, у кого есть ссылка»).
   - Нажмите **Развернуть** → скопируйте **URL** веб‑приложения.
5) В коде сайта замените строку:
   var ENDPOINT = window.CFG_ENDPOINT || 'WEB_APP_URL';
   на ваш реальный URL либо задайте window.CFG_ENDPOINT в блоке <script> до основного кода.
6) Залейте сайт на Vercel. Тест: откройте любой язык c UTM (?utm_source=test...), отправьте форму.
   Должно: строка в Google Sheets «Лиды C|F|G consulting», письмо на почту и, если настроен бот, сообщение в Telegram.
